# Sora Profile Archiver

A lightweight Python tool to archive videos from a Sora profile using Selenium automation.

## Features
- **Browser Automation:** Uses Selenium to render the React-based frontend.
- **Auto-Scroll:** Automatically scrolls the page to trigger lazy-loading of videos.
- **Session Auth:** Supports `__Secure-next-auth.session-token` for authenticated access.
